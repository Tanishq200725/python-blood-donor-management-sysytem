Blood Donation Management System

A Blood Donation Management System is a Python-based application designed to manage blood donors, recipients, and blood inventory efficiently. This system helps hospitals, blood banks, and organizations streamline the process of blood donation and request handling.

📌 Features
🔍 Search for donors by blood group
🧾 Add, update, and delete donor records
🩸 Manage blood inventory
📅 Record donation dates
🚫 Error handling for missing donors
📊 Dashboard (optional enhancements)
🌙 Dark mode (if GUI implemented)
📥 User-friendly interface (GUI-based)
🛠️ Tech Stack
Language: Python
Libraries Used:
Tkinter (GUI)
Pandas (Data handling)
Matplotlib (Visualization - optional)
Database: CSV / SQLite (depending on implementation)
